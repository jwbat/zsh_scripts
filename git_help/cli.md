## GitHub CLI

```sh
brew install gh
gh --version
gh auth login
gh auth status
```

```sh
gh repo clone jwbat/project-name        # clone a repository
gh repo clone ventacity/sbp-mono

gh repo create new-repo-name            # create a new repository
gh repo view owner/repo                 # view repository details
gh pr create --title "Pull Request Title" --body "Description of the pull request"
gh auth status                          # check the status of your authentication
```
